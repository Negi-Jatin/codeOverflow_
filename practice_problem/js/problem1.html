Convert Temperature Celcius to Farenhiet.

Problem: Write the JavaScript function to convert the Temperature value from Celcius to Farenhiet. Assume that the temperature value is 100

Solution: We can easily convert the temperature from Celcius to Farenhiet by multiplying the temperature(in celcius) by 9/5 and then add it with 32, it will provide you the desire temperature in Farenhiet.
